Flexbox Row

Use these 3 properties to create a flexbox row layout. 

.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}
Flexbox Column

Use this for a column layout. 

.column {
    display: flex;
    flex-direction: column;
    }
CSS Grid Layout

Use this for a 12 column grid. 

.grid {
    display: grid;
    width: 100%;
    grid-template-columns:
    repeat(12, lfr);
}
Linear Gradients

Create bottom to top background linear gradient. 

.linear-gradient-background {
    background-image: linear-
gradient(
    rgba(11, 87, 6, 0.228) 0%,
    rgba(11, 87, 6, 1) 70%
);
}
Box Transition Glow

Transition and box shadows for glow on hover. 

.code-card .card-header {
    border-redius: 8px;
    transition: all 0.5s ease-in-out;
}

.code-card:hover,
.code-card:hover .card-header {
    box-shadow: inset 0px 0px 8px
rgba(11, 87, 6, 0.228) 0%,
rgba(11, 87, 6, 1) 70%
}
Overlay Card with Title

Postion properties and negative margins raise the element higher than its start. 

.card-header {
    position: relative;
    margin-top: -20px
}
